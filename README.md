
Sistema Bancário em Python

Sobre o Projeto

Este projeto é um sistema bancário simples desenvolvido em Python. O sistema permite realizar operações básicas de uma conta bancária, como depósitos, saques, consulta de saldo e visualização de um resumo das movimentações realizadas.

O objetivo do projeto é praticar conceitos fundamentais de programação e lógica utilizando situações próximas de um sistema financeiro real.

Funcionalidades:

Consultar saldo atual da conta; 
Realizar depósitos; 
Realizar saques; 
Impedir saques com saldo insuficiente; 
Exibir resumo das movimentações; 
Contabilizar quantidade de depósitos realizados; 
Contabilizar quantidade de saques realizados;
Encerrar o sistema através de menu interativo; 


Tecnologias Utilizadas:

Python 3


Conceitos Praticados:

Variáveis; 
Operadores matemáticos; 
Estruturas condicionais (if, elif, else); 
Estruturas de repetição (while); 
Entrada e saída de dados; 
Formatação de valores monetários; 
Controle de fluxo; 
Validação de dados; 


Menu do Sistema:

1 - Verificar Saldo
2 - Realizar depósito
3 - Realizar Saque
4 - Ver Resumo
5 - Sair


Exemplo de Resumo:


===== RESUMO =====

Saldo atual: R$ 500.00
Total depositado: R$ 1000.00
Total sacado: R$ 500.00
Quantidade de depósitos: 3
Quantidade de saques: 2


Regras de Negócio:

Depósitos
Apenas valores maiores que zero são aceitos.
O saldo é atualizado após cada depósito.
Saques
O valor do saque não pode ser maior que o saldo disponível.
O saldo é atualizado após cada saque.


Resumo:

Exibe o saldo atual.
Exibe o total movimentado em depósitos.
Exibe o total movimentado em saques.
Exibe a quantidade de operações realizadas.
